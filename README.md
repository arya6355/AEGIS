# PROJECT AEGIS

## Autonomous Emergency Grid & Integrated Shelter

Project AEGIS is a low-cost, solar-powered smart emergency booth designed for flood-prone villages.

It provides:
- Early flood alerts
- Emergency communication
- GPS-based rescue support
- Autonomous safety monitoring
- Temporary shelter assistance

---

## Features

- ESP32-based automation
- Water level monitoring
- Smoke/Gas detection
- LED alert system
- Emergency buzzer siren
- Servo-controlled emergency door
- SOS button
- GPS rescue alert system
- Solar-powered operation

---

## Tech Stack

- ESP32
- Embedded C / Arduino IDE
- HC-SR04 Ultrasonic Sensor
- MQ Gas Sensor
- Servo Motor
- LEDs
- Buzzer
- Solar Panel + Battery
